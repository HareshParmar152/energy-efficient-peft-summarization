BEFORE YOU START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CHECKLIST:
✅ Virtual environment activated (.venv)
✅ PyTorch installed
✅ All dependencies installed
✅ Datasets available locally
✅ GPU with sufficient memory (8+ GB recommended)

CHECK GPU MEMORY:
```
python -c "import torch; print(f'GPU Memory: {torch.cuda.get_device_properties(0).total_memory / 1e9:.2f} GB')"
```

If less than 6 GB:
⚠️ Experiments may fail
⚠️ Consider reducing batch size further
⚠️ Or upgrade GPU

├─ Set CUDA_LAUNCH_BLOCKING=1 (already done)
├─ Restart GPU
├─ Reduce batch size
└─ Check GPU memory

├─ Check output directory permissions
├─ Ensure sufficient disk space
├─ Verify training completed
└─ Check logs for errors
