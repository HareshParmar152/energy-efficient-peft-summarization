================================================================================
WHAT HAPPENS DURING TRAINING
================================================================================

1. Energy tracking starts (CodeCarbon)
2. Dataset loads from local disk
3. Model loads (with quantization for QLoRA)
4. PEFT configuration applies
5. Training runs for 2000 steps
6. Energy tracking stops
7. Model saves to checkpoint directory
8. Results JSON saves with all metrics
9. Success/failure message displays

================================================================================
IF TRAINING FAILS
================================================================================

The script will:
1. Stop energy tracking
2. Save failure JSON with error details
3. Display error message
4. Exit gracefully

You can then:
- Check the *_FAILED.json file for error details
- Move to the next experiment
- Document the failure (hardware limitation)
